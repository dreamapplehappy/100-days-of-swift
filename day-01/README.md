
## Notes:

### 
+ 变量声明使用`var`
+ 字符串表示使用双引号`"`
+ 语句后面不需要带分号`;`

### Strings and integers
+ 快捷键`⇧↵`表示运行到当前行
+ 快捷键`⇧⌘↵`表示运行整个`playground`
+ 关于数字千位分隔符 TODO

### Multi-line strings
+ 多行字符串的`"""`开始和结束符号各占单独的一行
+ 如果多行字符串里面的内容不需要换行，需要在每行的结尾添加反斜线

### Constants
+ 使用let声明常量，这一点跟 `JavaScript` 不一样
+ 使用`⌥⇧X`删除一行
